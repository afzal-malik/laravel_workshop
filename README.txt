# Inventory System

## Introduction

This is a simple Inventory CRUD application which is built using PHP and the Laravel framework inorder to manage an inventory system.
the CRUD Application allows users to add,edit,delete,view and list the details of the items in the inventory.

## Features available in the Inventory System

- **Item List**: Display a list of all items in the inventory.
- **Add Item**: Add a new item to the inventory.
- **Edit Item**: Edit the details of an existing item in the inventory.
- **Delete Item**: Remove an item from the inventory.
- **View Item**: View details of a specific item in the inventory.

## Requirements

- PHP 8.1 or latest versions
- Composer
- MySQL or any other compatible database
